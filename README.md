# math.js
> A simple math library written in Javascript

## Install
```
$ npm install math.js
```

## Use
```javascript
// load the module
var math = require('math.js');

// ask for a number's ceil
math.ceil(13);
```

## Test
To execute the library's tests, first install the project dependencies:

```
$ npm install
```

Then, run the tests
```
$ npm test
```
